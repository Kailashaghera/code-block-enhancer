Add a "Copy" button and full color customization options for code blocks in your WordPress posts.

== Description ==

This plugin enhances the WordPress core/code block by:

* Adding a "Copy" button for code snippets
* Allowing full customization of:
  - Code block background color
  - Code text color
  - Border color
  - Text selection background & color
  - Copy button background, text color, and hover states

All options are configurable from the WordPress admin settings panel under "Settings > Code Block Customizer".

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/code-block-enhancer` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Go to "Settings > Code Block Customizer" to configure styles.

== Frequently Asked Questions ==

= Does this support syntax highlighting? =
No. This plugin only styles the default code blocks and adds a copy button. For syntax highlighting, use with plugins like "PrismJS" or "SyntaxHighlighter".

= Will it affect other themes? =
This plugin uses specific selectors and `!important` flags to ensure compatibility with most themes.
